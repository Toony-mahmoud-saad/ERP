يمكنك الآن الوصول إلى واجهات API التالية:

المصادقة: تسجيل الدخول، تسجيل مستخدم جديد (للمشرفين فقط)

إدارة الموظفين: إضافة/تعديل/حذف/عرض الموظفين

الحضور والانصراف: تسجيل الحضور والانصراف، عرض سجلات الحضور

الإجازات: طلب إجازة، الموافقة على الإجازات، عرض طلبات الإجازة

الرواتب: إنشاء كشوف الرواتب، عرض الرواتب، تقارير الرواتب

ملاحظات
تم تضمين نظام صلاحيات كامل حيث:

المشرف (admin): يمكنه فعل كل شيء

موظف الموارد البشرية (hr): يمكنه إدارة الموظفين والحضور والإجازات والرواتب

الموظف العادي (employee): يمكنه فقط عرض بياناته الشخصية وتسجيل الحضور والانصراف وطلب الإجازات

تم تضمين جميع الميزات المطلوبة:

إدارة ملفات الموظفين

حساب الرواتب الشهرية مع خصومات الغياب

تتبع الحضور والانصراف

متابعة الإجازات

تقارير الموارد البشرية

يمكنك توسيع النظام بإضافة المزيد من الميزات مثل:

إرفاق مستندات للموظفين

إشعارات عبر البريد الإلكتروني

واجهة مستخدم رسومية

تقارير أكثر تفصيلاً
